# TwiQ

## ディレクトリ

- src
    - assets : js, css, imageなど
    - controllers : コントローラー
    - functions
        - config.php : 全体設定
        - function.php : 関数(サブルーチン)の定義
        - keys.php : 環境ごとの設定
        - oauth.php : twitteroauth.phpで用いられる
        - twitteroauth.php : TwitterAPI利用ライブラリ
    - uploads : 画像のアップロード用ディレクトリ
    - views : ビュー
    - .htaccess
    - routes.php : ルーター
- init.sql : テーブルの定義
- readme.md : このファイル
